# pxq
react + react-router + redux + immutable + less + ES6 + webpack + fetch + 按需加载


### 下载

  git clone http://code.putao.io/web/weixin_orders.git

  cd weixin_orders

  npm install 

  npm install webpack -g 


### 运行（nodejs 6.0+）

  npm run dev (开发版本访问：http://localhost:8080/)
  
  npm run dist （发布生产版本）
